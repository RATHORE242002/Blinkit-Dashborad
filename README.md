# Blinkit-Dashborad

🛒 Blinkit Sales Analysis & Performance Dashboard

📌 Project Overview

This project is a Blinkit Grocery Data Analysis and Performance Dashboard developed using Power Bi
The dashboard helps in analyzing sales, item types, outlet performance, and customer trends using an interactive visual interface.

The dataset used contains grocery product details such as item type, fat content, outlet size, location, visibility, and sales.

🎯 Objectives
To analyze Blinkit grocery sales data
To visualize outlet performance
To understand item category trends
To build an interactive dashboard using Django
To perform data-driven decision making

📈 Dashboard Features
📌 Total Sales Overview
📌 Sales by Item Type
📌 Sales by Outlet Size
📌 Sales by Outlet Location (Tier-wise)
📌 Item Fat Content Analysis
📌 Outlet Performance Comparison
📌 Interactive Charts & Graphs

📊 Dataset Description
The dataset includes the following features:
Item Identifier
Item Fat Content (Low Fat, Regular)
Item Type (Frozen Foods, Dairy, Soft Drinks, etc.)
Outlet Identifier
Outlet Size (Small, Medium, High)
Outlet Location Type (Tier 1, Tier 2, Tier 3)
Outlet Type (Supermarket Type1, Type2, etc.)
Item Visibility
Item Weight
Sales
Total Records: 8523 Rows (Blinkit Grocery Dataset)

📊 Key Insights (Example)
Tier 3 outlets have higher sales compared to Tier 1
Medium-sized outlets contribute the most to revenue
Low Fat items dominate in sales volume
Supermarket Type1 shows highest performance
